# Hyper Key System Cheat Sheet

**Hyper = Caps Lock = Ctrl+Opt+Cmd (no Shift)**
**Tap Caps Lock = Escape**

## Navigation

| Keys | Action | Layer |
|------|--------|-------|
| `Hyper+HJKL` | Focus window | Aerospace |
| `Hyper+Shift+HJKL` | Focus pane | Zellij (in Ghostty) |
| `Hyper+1-9` | Switch workspace | Aerospace |
| `Hyper+Shift+1-9` | Switch tab | Zellij (in Ghostty) |

## Core Actions

| Keys | Action |
|------|--------|
| `Hyper+Space` | Toggle last app |
| `Hyper+P` | Raycast |
| `Hyper+Enter` | Open Ghostty |
| `Hyper+[/]` | Prev/Next tab |
| `Hyper+M` | Aerospace move mode |
| `Hyper+R` | Aerospace resize mode |

## Zellij Actions (in Ghostty)

| Keys | Action |
|------|--------|
| `Hyper+Shift+T` | New tab |
| `Hyper+Shift+N` | New pane |
| `Hyper+Shift+X` | Close pane |
| `Hyper+Shift+W` | Toggle floating |
| `Hyper+Shift+F` | Fullscreen pane |
| `Hyper+Shift+S` | Session manager |
| `Hyper+Shift+Arrows` | Resize pane |
